# Start App

## Clean - optional

- clean if PG creds changed

```bash
yarn clean:minerva
```

## Initalize Logto Apps

```bash
yarn init:logto
```

## Start UI

```bash
yarn start:ui --wait
```

## Start App

```bash
yarn start:minerva --wait
```

## All-In-One

```bash
yarn clean:minerva && yarn init:logto && yarn start:ui --wait && yarn start:minerva --wait
```
